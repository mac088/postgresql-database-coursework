# postgresql-database-coursework
Курсовая работа: Исследование и реализация SQL-запросов в PostgreSQL

Описание проекта:
Данная курсовая работа посвящена изучению и практической реализации SQL-запросов различного уровня сложности в системе управления базами данных PostgreSQL. Проект демонстрирует создание и взаимодействие между связанными таблицами, реализацию триггеров, индексов, представлений, функций, процедур и ролей пользователей.

Основные задачи проекта:

Проектирование и создание структуры базы данных (рецепты, ингредиенты, шаги, технологии).

Реализация сложных SQL-запросов: подзапросы, JOIN, CASE, агрегирование, ANY/ALL, коррелированные запросы.

Создание VIEW и материализованных представлений, включая обновляемые представления с использованием триггеров и правил.

Оптимизация запросов с помощью индексов разных типов (BRIN, B-tree, SPGiST) и анализ планов выполнения.

Разработка триггеров и триггерных функций для логирования изменений и валидации данных.

Создание хранимых процедур и функций для добавления, обновления и удаления записей.

Реализация транзакций, курсов, скалярных и векторных функций.

Настройка ролей и прав доступа пользователей (администратор и посетитель).

Ведение исторических данных (OLAP) для отслеживания изменений стоимости ингредиентов.

Технологии:

PostgreSQL

SQL / PLpgSQL

Результат:
Проект демонстрирует полное владение инструментами PostgreSQL для работы с реляционными данными — от проектирования структуры и написания запросов до оптимизации и обеспечения безопасности. Может использоваться как учебный пример или шаблон для разработки серверной логики приложений с базами данных PostgreSQL.
